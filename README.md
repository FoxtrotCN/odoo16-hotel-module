# 🏨 Odoo 16 - Módulo Hotel

Módulo de gestión hotelera desarrollado sobre Odoo 16 como proyecto de aprendizaje.

## Funcionalidades (en desarrollo)
- Gestión de habitaciones
- Gestión de huéspedes
- Reservas con check-in y check-out
- Wizard de facturación

## Stack
- Odoo 16
- PostgreSQL 14
- Docker

## Instalación local
1. Clonar el repositorio
2. Ejecutar `docker compose up -d`
3. Instalar el módulo `Hotel` desde el panel de aplicaciones
